# Your first form

Creating your first form and understanding FormKit.

## Introduction

Let's start by creating our first FormKit form! We'll learn some of FormKit's key features and how they benefit you. We'll also pick up some nice tips along the way — like how to manage form state without relying on <code>modelValue</code> and <code>onUpdateModelValue</code>.

> **Info:** This guide assumes you are familiar with your framework’s component model, props, and event handlers.
